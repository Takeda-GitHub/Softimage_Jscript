このスクリプトはモーションチェック用にカメラを作るスクリプトになります。
よくモーションでグローバルに移動値を持っていると、確認し辛い事があるかと思います。

解決策としてカメラを新規で作成し、その親をCOGにコンストをかけると確認し易い
環境ができるかと思います。

このスクリプトはそれを自動化したものになります。
使い方は下記の通り↓

表示したいビューをアクティブにした後
グローバルの移動値を持っているコントローラー(COG等)を選択し、実行↓


これでアクティブなビューがこのカメラに変更されます。
また設定を変えたい場合は
スクリプトの
「Translate(Cam, 50, 50, 50, siRelative, siView, siObj, siX, null, null, null, null, null, null, null, null, null, 0, null);」
辺りを変更することで簡単にできます！
